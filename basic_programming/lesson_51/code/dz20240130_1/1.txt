Дан List<Programmer>. Programmer {name, List<Task>, String city} Task - класс, который состоит из {int id, String title, int days, Status status }, Status: DONE,
PROCESSED, CANCELED.   Причем, одна задача может находиться в списках у нескольких программистов. За задачу отвечают все, у кого она в списке.

Необходимо:
Написать метод, который вернет список всех задач в работе (статус DONE) , отсортированый по количеству дней в работе (от большего к меньшему)

Написать метод, который вернет MAP<Boolean, List<Task>>, где ключу true соответствует List задач в состояние DONE, а ключу false - остальные


Написать метод, который вернет MAP где ключ - задача, а значение - список ответственных за нее программистов.
